Quick Start Guide

  1. Option 1: Python Server
    Open terminal in this folder and run:
    python -m http.server 8000
    Then visit: http://localhost:8000
  2. Option 2: Node.js Server
    Run: npx serve
    Follow the URL shown in terminal
  3. Option 3: Direct Open
    Click the "Launch Application" button above
    (May have limited functionality)
